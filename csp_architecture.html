SSE //
Semantic State Extractor
Reads a conversation and distills five semantic vectors: intent, reasoning chain, entity graph, uncertainty map, and momentum. Model-agnostic output.
USC //
Universal Semantic Compressor
Maps model-specific internal representations to a shared embedding space. Target: 50× reduction. Backbone: small universal encoder trained across model families.
FC //
Fidelity Certificate
Signed proof of how much semantic content survived compression. Attached to every state packet. Receiving model knows exactly what was preserved.
SR //
State Receiver
Takes a CSP packet and generates a cognitive primer — a specially formatted context that reconstructs the semantic state inside the receiving model's architecture.
SCS //
Semantic Continuity Score
The grand challenge metric. Measures how accurately a receiving model continues a reasoning thread. Validated against human preference ratings. This is the GDT score for thought transfer.
ZKV //
ZK Continuity Proof
Receiving model proves it actually used the CSP packet rather than starting fresh. Prevents cognitive fraud. Extends SMART-UAHP's Selective Attention Sample proofs.

"CSP defines a Semantic Continuity Score (SCS) metric for cross-model thought transfer fidelity, targeting ≥0.85 across heterogeneous model families. By compressing the five canonical semantic state vectors — intent, reasoning chain, entity graph, uncertainty map, and momentum — to a 50× reduced universal embedding, CSP enables any receiving model to reconstruct and continue a reasoning thread without access to the original conversation history. Combined with UAHP v0.5.4 transport and SMART-UAHP substrate routing, this constitutes the first complete protocol stack for substrate-agnostic, portable, cryptographically-verified intelligence flow."
